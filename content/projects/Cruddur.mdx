# Cruddur Project

Cruddur is a microblogging social media platform designed to demonstrate cloud-native development principles, including microservices, containerization, and serverless patterns.

![Cruddur Project Image](/cruddur.webp)

## Overview (Cloud Engineer Perspective)

The Cruddur project showcases a modern, distributed application architecture. It typically involves a frontend served by a static site generator or a lightweight framework, communicating with a set of independent microservices for user management, post creation, and feed generation. These services are deployed as containers on a managed Kubernetes service or serverless functions, utilizing API Gateway for routing and authentication. Data is persisted across various specialized databases, optimized for each service's needs.

## Features

-   **User Authentication & Profiles:** Secure user registration and profile management.
-   **Post Creation & Feed:** Users can create short posts and view personalized feeds.
-   **Microservices Architecture:** Decoupled services for scalability and independent development.
-   **Real-time Updates:** Potentially uses WebSockets or server-sent events for live content.

## Technology Stack

-   **Frontend:** React, Next.js
-   **Backend:** Node.js (Express/Fastify), Python (FastAPI) for microservices
-   **Containerization:** Docker, Kubernetes (EKS/GKE/AKS)
-   **Databases:** PostgreSQL (for relational data), DynamoDB (for key-value), Redis (for caching)
-   **Messaging:** Kafka, RabbitMQ (for inter-service communication)
-   **CI/CD:** GitHub Actions, GitLab CI

## Importance

Cruddur serves as an excellent example of building scalable and resilient cloud applications. It demonstrates how to break down a monolithic application into manageable microservices, deploy them efficiently using container orchestration, and leverage various cloud services for optimal performance and cost-effectiveness. This project is valuable for understanding modern application development and deployment strategies in a cloud environment.
