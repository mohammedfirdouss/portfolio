TerraTowns is a community project inspired by the self-hosted spirit of Geocities. Developers ("Terraformers") use Infrastructure as Code to launch their own personal websites, called "Terra Houses." The project started as a hands-on Terraform bootcamp, where each user writes Terraform code to provision and deploy a simple CMS that becomes part of the TerraTowns network, backed by a custom Go provider, AWS (S3, CloudFront, Lambda), and a Ruby mock server for local development.
