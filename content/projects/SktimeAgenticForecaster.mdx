Sktime Agentic Forecaster is a Python project that accepts a forecasting task in plain English, selects suitable `sktime` estimators, builds a pipeline, fits it on data, and returns predictions plus explanation metadata.

The goal is to make time-series experimentation faster without hiding the underlying forecasting objects. The output is a real sktime pipeline that can be inspected, reused, and refined.

## What it is

A forecasting assistant focused on practical experimentation:

- **Natural-Language Forecasting:** Describe the task in text and get a full forecasting run.
- **Automatic Pipeline Selection:** Chooses from curated forecasters and transformers.
- **Structured Output:** Returns predictions, selected estimators, pipeline object, and explanation.
- **Backend Choice:** Works with Anthropic, OpenAI, Gemini, or LangChain-compatible models.

## How It's Built

- **Core Language:** Python with sktime as the forecasting engine.
- **Agent Flow:** Prompt parsing, data-context extraction, estimator selection, pipeline build, fit, predict.
- **Pipeline Output:** Produces a live `TransformedTargetForecaster` object.
- **Testing Strategy:** Mocked LLM calls for deterministic tests without external API keys.
- **Developer Experience:** Extras-based installs for backend-specific dependencies.

## What I Learned

- **Agent Design for ML:** Translating vague text requests into concrete model-building steps.
- **Forecasting Trade-offs:** Choosing transformations and forecasters based on data characteristics.
- **Model-Agnostic Integration:** Designing a clean interface across multiple LLM providers.
- **Explainability:** Pairing forecasts with concise rationale to support model trust and iteration.
