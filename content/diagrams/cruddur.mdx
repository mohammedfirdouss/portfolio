# Cruddur

This section provides a technical diagram and explanation of the Cruddur microblogging platform, highlighting its simplicity, scalability, and modern technology stack.

## Features
- User authentication and profiles
- Real-time updates
- Scalable backend using modern cloud technologies

## Technology Stack
- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Database**: PostgreSQL

## Deployment
Cruddur is deployed using containerized services and managed through Kubernetes for high availability and fault tolerance.
